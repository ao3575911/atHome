# Demo Fixtures — Local Development Only

The manifest files in this directory (`*.json`) are **local development samples generated by `npm run demo`**. They are not production identity roots.

- Keys are demo-only Ed25519 keypairs with no real-world authority.
- Any private keys previously associated with these manifests are treated as compromised samples and must not be reused.
- Do not use these manifests as the basis for a real production deployment.

For production use, generate new identities with client-side or KMS/HSM key custody per the [Production Key Custody Plan](../../docs/key-custody-plan.md).
